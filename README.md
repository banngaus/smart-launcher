**SmartLauncher** — это ультимативный хаб для автоматизации Windows. Выполняйте Python/PowerShell скрипты, конвертируйте медиафайлы, устанавливайте софт и держите систему в чистоте через красивый и современный интерфейс. 

Вам больше не нужно ковыряться в системных переменных (PATH), устанавливать Python или качать FFmpeg вручную — **всё уже работает из коробки**.

---

## ✨ Главные фичи

### 🛠 1. Менеджер скриптов (Zero-Install)
Выполняйте свои и встроенные `.py`, `.bat` и `.ps1` скрипты в один клик.
*   **Встроенный Python:** Приложение использует Embedded Python. Ваши скрипты будут работать даже на чистой Windows.
*   **Чтение логов (Live Console):** Вывод консоли в реальном времени прямо в красивом всплывающем окне внутри лаунчера.
*   **Передача аргументов:** Возможность настроить скрипт так, чтобы лаунчер сам просил пользователя выбрать файл, папку или ввести текст перед запуском.
*   **Встроенные задачи:** Пакетный ресайз фото, создание GIF, извлечение аудио, сортировка загрузок и многое другое.

### 🔄 2. Универсальный медиа-конвертер
Встроенный GUI для FFmpeg (сам FFmpeg зашит в приложение).
*   **Видео:** MP4, MKV, AVI, MOV, WEBM (с поддержкой выбора качества).
*   **Аудио:** MP3, WAV, FLAC, AAC, OGG.
*   **Изображения:** JPG, PNG, WEBP, ICO (удобно для создания иконок).
*   *Поддержка Drag & Drop и пакетной обработки.*

### 📦 3. Каталог приложений (Тихая загрузка)
Забудьте про поиск софта в интернете после переустановки Windows.
*   Скачивайте базовый софт (7-Zip, Chrome, VLC, VS Code, Git и др.) прямо из лаунчера.
*   Автоматическая загрузка установщиков в `%TEMP%` с отображением прогресс-бара.
*   Установка необходимых системных компонентов (.NET 8, Visual C++ All-in-One, DirectX) в один клик.
*   Умное сканирование реестра: лаунчер сам знает, что у вас уже установлено.

### 🧹 4. Глубокая очистка системы
Освобождайте гигабайты места без сторонних "клинеров" с закрытым кодом.
*   13 модулей очистки: кэш обновлений Windows, временные файлы (Temp), корзина, Prefetch, дампы памяти (MEMORY.DMP).
*   Очистка кэша браузеров (Chrome, Edge, Firefox), DNS-кэша, кэша иконок и шрифтов.
*   Удаление старых точек восстановления системы.

### ☁️ 5. Автообновление через GitHub
Лаунчер сам проверяет релизы на GitHub. Если есть новая версия, он покажет список изменений, скачает инсталлер, закроет старую версию, обновится и запустится заново.

## 📥 Как начать пользоваться?

1. Перейдите в раздел [Releases](../../releases/latest).
2. Скачайте файл `SmartLauncher_Setup_X.X.X.exe`.
3. Запустите (установка быстрая, права администратора не требуются для установки в AppData).
4. Готово! Приложение появится на рабочем столе и в меню Пуск.
